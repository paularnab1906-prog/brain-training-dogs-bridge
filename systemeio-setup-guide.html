Systeme.io Funnel Setup Guide

Complete walkthrough to connect your opt-in page, thank-you page, and email sequence for the Brain Training for Dogs affiliate funnel.

Files You Have

systemeio-optin.html
Opt-in / squeeze page — the page visitors see first. Contains the email capture form.
systemeio-thankyou.html
Thank-you page — shown after email submission. Confirms delivery + offers next step.

Step 1 — Create the Funnel

1 Create a new funnel in Systeme.io
  1. Log into Systeme.io
  2. Click Funnels in the left sidebar
  3. Click Create (top right)
  4. Name it: Brain Games Lead Magnet
  5. Choose Build an audience as the goal
  6. Click Create

Step 2 — Set Up the Opt-in Page

2 Configure the opt-in page
  1. You will see one step: Opt-in page — click on it
  2. Click Edit
  3. Choose a template or start blank
  4. Option A — Use Systeme.io builder: Design the page using drag-and-drop blocks. Add a headline, image, and form block.
  5. Option B — Paste custom HTML: Add a "Custom HTML" block and paste the contents of systemeio-optin.html inside it.
  6. Make sure the form has an email input field (Systeme.io adds this by default)
  7. Click Save
Important: If you use the custom HTML file, replace the placeholder form action URL with your actual Systeme.io endpoint. Or just use Systeme.io's built-in form block and delete the HTML form entirely — Systeme.io handles the subscription automatically.

Step 3 — Set Up the Thank-You Page

3 Add the thank-you page
  1. Click Add step below the opt-in page
  2. Choose Thank you page
  3. Click Edit
  4. Add a Custom HTML block and paste the contents of systemeio-thankyou.html
  5. Or design it with the builder: headline + confirmation message + optional download button
  6. Click Save

Step 4 — Create the Email Automation

4 Set up the autoresponder sequence

4a. Create an automation rule

  1. Go to Emails → Campaigns in the left sidebar
  2. Click Create → choose Automation rule
  3. Set trigger: When a contact subscribes to funnel [Brain Games Lead Magnet]
  4. Set action: Send email

4b. Create the delivery email

  1. Go to Emails → Emails
  2. Click Create
  3. Subject line: Your 5 Brain Games PDF is here 🐕
  4. Body template:
Hi {contact_first_name},

Your free PDF is ready!

Download it here: [YOUR-PDF-LINK]

Inside you will find 5 brain games that take just 5 minutes each.
Most owners see a calmer dog by the end of the day.

If you want the full program with 21 games and video lessons,
here is the link: [YOUR-CLICKBANK-HOPLINK]

To your dog's happiness,
Smart Paw Tips
  • Attach the PDF or include a download link
  • Click Save and publish

    • Step 5 — Connect Your Domain (Optional but Recommended)

    5 Point your domain to Systeme.io
    1. Buy a domain (e.g. smartpawtips.com) — ~$10/year
    2. In Systeme.io, go to Settings → Domains
    3. Add your domain and follow the DNS instructions
    4. Set the funnel URL to something like smartpawtips.com/brain-games
    Why this matters: A branded domain looks more trustworthy than a systeme.io/... URL. Higher trust = higher opt-in rate.

    Step 6 — Test the Full Flow

    6 Verify everything works
    1. Open your opt-in page URL in an incognito browser
    2. Enter a test email address
    3. Submit the form
    4. Verify you land on the thank-you page
    5. Check your inbox for the delivery email
    6. Confirm the PDF link works
